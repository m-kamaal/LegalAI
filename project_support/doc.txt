GOAL:
To help advocates build there cases faster without any hallucination by collecting the material via online books, recent case results, websites and other authentic sources.

PROBLEM:
Advocates require alot of time to collect reference and make their cases, or if they use chatgpt, there is a ton of hallucination that they cant review

USER STORIES:
1- User should be able to upload a document in any format and ask questions from it.
2- User should be able to get information of a specific case from website (need more details about websites)
3- User should be able to generate answers in a formal law language to be dispalyed in front of judges.
4- User should get actual references to all the points generated by AI so they can cross check quickly

DEFINE DATA MODELS:
1- tax websites
2- Law books
3- Case judgements and notifications
